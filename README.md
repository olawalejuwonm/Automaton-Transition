# Automaton Transition Generator

This project generates a simple boolean automaton based on user-defined transition relations.

## Getting Started

Install dependencies:

```powershell
npm install
```  

Compile the project:

```powershell
npm run build
```  

Run the generator:

```powershell
npm start -- <transitions file>
npm start -- transitions.txt
```

## Requirements

- Node.js 14+
- TypeScript
